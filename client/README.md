
# Frontend - Lessons Catalog App

This React-based frontend allows users to search, filter, and manage lessons fetched from the backend API. It features a modern UI with Tailwind CSS styling.

---

## Features

- **Search**: Search lessons by title.
- **Filter**: Filter lessons by tags (e.g., Math).
- **Display**: View lessons with images, descriptions, and duration.
- **Select**: Select and deselect lessons.
- **Responsive**: Fully responsive design for all devices.

---

## Requirements

- **Node.js** (v14+)
- **npm** (v6+)

---

## Setup

### 1. Install Dependencies

Run the following command to install all necessary dependencies:

```bash
npm install
```

### 2. Update API URL (if different)

Ensure your backend server is running at `http://localhost:5000`, or update the API URL in your code accordingly.

---

## Notes

- Tailwind CSS is used for styling.
- You can customize styles and components as needed.
